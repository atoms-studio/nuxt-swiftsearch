---
title: "<AisConfigureRelatedItems>"
description: Renderless helper that configures related-items search parameters.
---

## Usage

```vue
<AisInstantSearch :configuration="configuration">
  <AisConfigureRelatedItems
    :hit="referenceHit"
    :matching-patterns="{
      brand: { score: 3 },
      categories: { score: 2 },
    }"
  />
  <AisHits />
</AisInstantSearch>
```

## Props

| Prop                        | Type                                | Default     | Description                                     |
| --------------------------- | ----------------------------------- | ----------- | ----------------------------------------------- |
| `hit`                       | `AlgoliaHit`                        | required    | Reference hit used to derive optional filters.  |
| `matchingPatterns`          | `Record<string, { score: number }>` | required    | Matching schema per attribute.                  |
| `transformSearchParameters` | `function`                          | `undefined` | Final transform of generated search parameters. |
