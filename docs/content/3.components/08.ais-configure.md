---
title: "<AisConfigure>"
description: Injects InstantSearch search parameters without rendering UI.
---

## Usage

```vue
<AisInstantSearch :configuration="configuration">
  <AisConfigure :hits-per-page.camel="12" query="apple" />
</AisInstantSearch>
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `searchParameters` | `Record<string, unknown>` | `{}` | Optional object payload for connector params. |
| `...any` | `unknown` | `undefined` | Additional props are forwarded as configure parameters (for example `query`, `ruleContexts`, `hitsPerPage`). |

::note
`AisConfigure` is intentionally UI-less. Use slots only if you need direct access to `refine` and current `searchParameters`.
::
