---
title: "<AisPoweredBy>"
description: Displays the Algolia attribution link.
---

## Usage

```vue
<AisInstantSearch :configuration="configuration">
  <AisPoweredBy theme="light" />
</AisInstantSearch>
```

## Props

| Prop    | Type                | Default           | Description                                |
| ------- | ------------------- | ----------------- | ------------------------------------------ |
| `theme` | `"light" \| "dark"` | `"light"`         | Visual variant class.                      |
| `url`   | `string`            | connector default | Optional custom attribution URL.           |
| `id`    | `string`            | `""`              | Optional widget ID for repeated instances. |
