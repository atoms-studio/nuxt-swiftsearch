---
title: "<AisBreadcrumb>"
description: Hierarchical breadcrumb navigation from facet paths.
---

## Usage

```vue
<AisInstantSearch :configuration="configuration">
  <AisBreadcrumb :attributes="['categories.lvl0', 'categories.lvl1']" />
</AisInstantSearch>
```

## Props

| Prop             | Type       | Default           | Description                                             |
| ---------------- | ---------- | ----------------- | ------------------------------------------------------- |
| `attributes`     | `string[]` | required          | Facet path attributes used to build breadcrumb levels.  |
| `separator`      | `string`   | connector default | Separator used by the connector when parsing levels.    |
| `rootPath`       | `string`   | `undefined`       | Optional root path prefix for hierarchical data.        |
| `transformItems` | `function` | `undefined`       | Transforms breadcrumb items before rendering.           |
| `id`             | `string`   | `""`              | Optional widget ID for repeated instances in one index. |
