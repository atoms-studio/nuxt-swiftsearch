---
title: "<AisQueryRuleCustomData>"
description: Displays custom data returned by Algolia query rules.
---

## Usage

```vue
<AisInstantSearch :configuration="configuration">
  <AisConfigure query="my-query" :rule-contexts.camel="['my-rule-context']" />

  <AisQueryRuleCustomData>
    <template #item="{ item }">
      <a :href="item.link">{{ item.title }}</a>
    </template>
  </AisQueryRuleCustomData>
</AisInstantSearch>
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `transformItems` | `function` | `undefined` | Transforms rule payload items before rendering. |
