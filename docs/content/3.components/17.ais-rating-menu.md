---
title: "<AisRatingMenu>"
description: Rating-based facet refinement (for example, stars and up).
---

## Usage

```vue
<AisInstantSearch :configuration="configuration">
  <AisRatingMenu attribute="rating" :max="5" />
</AisInstantSearch>
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `attribute` | `string` | required | Numeric attribute used as rating score. |
| `max` | `number` | `5` | Max rating scale for visual rendering. |
