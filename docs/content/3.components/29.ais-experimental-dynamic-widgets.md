---
title: "<AisExperimentalDynamicWidgets>"
description: Backward-compatible alias of `<AisDynamicWidgets>`.
---

`AisExperimentalDynamicWidgets` is kept for parity with vue-instantsearch naming.

Prefer `<AisDynamicWidgets>` for new code.

## Usage

```vue
<AisInstantSearch :configuration="configuration">
  <AisExperimentalDynamicWidgets>
    <AisRefinementList attribute="brand" />
    <AisRefinementList attribute="categories" />
  </AisExperimentalDynamicWidgets>
</AisInstantSearch>
```
