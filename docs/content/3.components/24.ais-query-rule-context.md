---
title: "<AisQueryRuleContext>"
description: Renderless widget that injects query rule contexts from tracked filters.
---

## Usage

```vue
<AisInstantSearch :configuration="configuration">
  <AisQueryRuleContext
    :tracked-filters="{
      brand: (values) => values.slice(0, 1),
    }"
  />

  <AisQueryRuleCustomData />
</AisInstantSearch>
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `trackedFilters` | `Record<string, (values) => values>` | required | Maps facet names to context extractor functions. |
| `transformRuleContexts` | `function` | `undefined` | Final transform for computed rule context values. |
