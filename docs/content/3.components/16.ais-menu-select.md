---
title: "<AisMenuSelect>"
description: Select-based variant of the menu connector.
---

## Usage

```vue
<AisInstantSearch :configuration="configuration" :widgets="widgets">
  <AisMenuSelect attribute="categories" />
</AisInstantSearch>

<script setup lang="ts">
const widgets = computed(() => [useAisMenu({ attribute: "categories" })]);
</script>
```

## Props

| Prop             | Type                    | Default           | Description                          |
| ---------------- | ----------------------- | ----------------- | ------------------------------------ |
| `attribute`      | `string`                | required          | Facet attribute name.                |
| `limit`          | `number`                | connector default | Number of menu options to fetch.     |
| `sortBy`         | `unknown[] \| function` | connector default | Sorting strategy for options.        |
| `transformItems` | `function`              | `undefined`       | Transforms options before rendering. |

::note
`AisMenuSelect` uses the `menu` connector state. Register `useAisMenu(...)` when using manual `:widgets` mode.
::
