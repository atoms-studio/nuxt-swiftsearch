---
title: "<AisRangeInput>"
description: Numeric range refinement with min/max inputs.
---

## Usage

```vue
<AisInstantSearch :configuration="configuration">
  <AisRangeInput attribute="price" :precision="2" />
</AisInstantSearch>
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `attribute` | `string` | required | Numeric attribute to refine. |
| `min` | `number` | connector default | Lower bound override. |
| `max` | `number` | connector default | Upper bound override. |
| `precision` | `number` | `0` | Decimal precision used for step calculation. |
