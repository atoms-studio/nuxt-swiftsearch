---
title: "<AisExperimentalConfigureRelatedItems>"
description: Backward-compatible alias for related-items configuration.
---

This component behaves like `<AisConfigureRelatedItems>` and is provided for vue-instantsearch parity.

## Usage

```vue
<AisInstantSearch :configuration="configuration">
  <AisExperimentalConfigureRelatedItems
    :hit="referenceHit"
    :matching-patterns="{ brand: { score: 3 } }"
  />
</AisInstantSearch>
```
