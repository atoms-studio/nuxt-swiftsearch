---
title: "<AisDynamicWidgets>"
description: Renders facet widgets based on facet ordering returned by Algolia.
---

## Usage

```vue
<AisInstantSearch :configuration="configuration">
  <AisDynamicWidgets>
    <AisRefinementList attribute="brand" />
    <AisRefinementList attribute="categories" />
    <AisMenu attribute="price_range" />
  </AisDynamicWidgets>
</AisInstantSearch>
```

## Props

| Prop                | Type                | Default           | Description                                               |
| ------------------- | ------------------- | ----------------- | --------------------------------------------------------- |
| `transformItems`    | `function`          | `undefined`       | Transforms `attributesToRender` before rendering widgets. |
| `facets`            | `['*'] \| string[]` | connector default | Facets requested for dynamic rendering.                   |
| `maxValuesPerFacet` | `number`            | connector default | Max facet values requested per facet.                     |
| `id`                | `string`            | `""`              | Optional widget ID for repeated instances.                |

::note
In manual `:widgets` mode, register `useAisDynamicWidgets(...)` and the nested child widgets explicitly.
::
