# nuxt-swiftsearch

Agent skill for implementing Algolia search UIs in Nuxt 3 apps using `@atoms-studio/nuxt-swiftsearch`.

## What this skill provides

- Setup and installation guidance
- Core dual-registration pattern (composable + component)
- Full component/composable reference table
- URL routing integration
- Slot customization patterns
- Caching strategies
- Common mistakes to avoid

## Usage

This skill is automatically triggered when working on search functionality with Algolia in Nuxt 3 projects that use `@atoms-studio/nuxt-swiftsearch`.
